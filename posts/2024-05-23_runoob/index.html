准备工作

Dev-C++（或者叫做 Dev-Cpp）是 Windows 环境下的一个轻量级 C/C++ 集成开发环境（IDE）。非常适合于C/C++语言初学者使用。

强烈建议在家里电脑上也安装此编程工具，方便以后复习、练习使用。具体步骤如下：

点击查看 - 搭建编程环境

如何下载

  1. 在自己的电脑上（台式、笔记本均可），登录官网下载址：https://sourceforge.net/projects/orwelldevcpp/
  2. 单击Download按钮，开始下载
  3. 等待下载过程（很快，不到一分钟；如果网差的话也没事，慢慢等几分钟即可搞定）
  4. 下载好后，找到安装包，名称叫Dev-Cpp 5.11 TDM-GCC 4.9.2 Setup.exe，即代表下载成功。

如何安装

双击Dev-Cpp 5.11 TDM-GCC 4.9.2 Setup.exe 安装包，开始下载，你将看依次看到如下界面：

  • 【第一步】Please wait while Setup is loading...unpacking data: 10% 此时代表是解压状态，不要进行操作，等待即可。

  • 【第二步】Installer Language — Please select a language 此时代表是选择语言，默认为English即可，不要进行修改，点击OK按钮，进入下一步

  • 【第三步】Dev-C++ 5.11 — License Agreement 此时代表相关协议，点击I Agree按钮，进入下一步

  • 【第四步】Dev-C++ 5.11 — Choose Components 此时代表组件的配置，不要进行其他操作，点击Next >按钮，进入下一步

  • 【第五步】Dev-C++ 5.11 — Choose Install Location 此时代表选择安装位置，怕麻烦的话也可不要进行其他操作，点击Install按钮即可，进入下一步

  • 【第六步】Dev-C++ 5.11 — Installing 此时代表着正在安装…，不要进行其他操作，耐心等待即可

  • 【第七步】Dev-C++ 5.11 — Completing the Dev-C++ 5.11 Setup Wizard 此时代表着安装成功，不要进行其他操作，点击Finish按钮，进入下一步

  • 【第八步】Dev-C++ 5.11 first time configuration — Select your language 此时代表初始配置语言，点击简体中文/Chinese后，再点击> Next按钮，进入下一步

  • 【第九步】Dev-C++ 5.11 first time configuration — 选择你的主题 此时代表自定义主题，不用进行其他操作，点击 > Next按钮，进入下一步

  • 最后，点击右下角 OK按钮，大功造成。

安装或使用过程中，遇到问题，可直接联系我

关于编程的那些事

我们都知道：生活中，笔是多种多样的，纸是有多种尺寸的，最通用的语言之一是英语。

编程的入门阶段，老师给你们选择的是最适合的。

学习之前心中有一个整体的把握，再学习其他编程，都是殊路同归。

编程就是写作

我一直认为编程就是写作，是与机器商量出一本书的过程。

  • 用什么笔？ 答：台式电脑、笔记本、平板…
  • 用什么纸？ 答：Dev-C++、Xcode、Visul Studio…
  • 怎么表达？ 答：C++语言、Python、Java、PHP….

怎么用笔？

  • 手机玩的这么溜，相信电脑也不在话下

怎么用纸？

  • 选择了Dev-C++工具，慢慢参考教程掌握，相信老师课堂上都教过

怎么表达？

  • 根据老师的安排，以RUNOOB教程网站为主，其他作为巩固

分享一些表达技巧，如下：

语文不能是体育老师教的

语文是基础学科，打从小就学。细细想一下，学的那些东西，不就是识字、背大量词语、掌握若干种语法吗？ 编程也一样：（以C++为例）

  • 识字是编程的基础：数据类型、关键字、变量常量、运算符…

    千万别问为什么int表示整数、bool表示true/false之类的问题。 我们是使用语言的人，不是创造者。问这些问题，就相当于问：汉字里的我的字形为什么是我，而不是长其他的模样。等自己超神一般杀戮的那一天，亲自创建一种编程语言，想怎么创造就怎么创造。

    学习过程中，我们要做的唯一一件事就是：记住，大量练习。

    如果非要刨根问底的话，编程语言里的好多关键字，都是英语的缩写 如：int 是 单词 integer的缩写；bool 是 单词 boolean的缩写； 另外，像for[从…] class[类] struct[结构] while[到…时] type[类别] template[模版] include[包含]，理解了单词的含义，会轻松很多。

    所以人家常说：英语好了，学得快。

  • 语法是算法的基础 我沉浸算法七年有余，深知数据结构的重要性。要想学好算法，必须得熟悉以下知识点：

    • 循环结构：for while…
    • 条件判断：if..else switch…
    • 数组、引用、变量： 这些知识点的概念必须要反复看，技巧的提升需要刷题练习。

  • 算法不是编程，是数学 切记算法重在思路，不在于编程语言。

    • 同样一种排序算法，可以用C++语言编写，也可以用JavaScript语言编写，甚至可以用汉字编写。算法的思路清晰了，语言只是一种表达方式，仅此而已。

    • 算法可以只用数组实现，也可以只用类来实现，也可以写一个函数实现…这就是所谓的编程风格。你熟悉哪一种，就用你擅长的。

    • 好的算法是无数前辈总结出来，流传至今。参加竞赛，必须要背诵各式各样的算法。算法相当于是内功心法，在不同的题目中表现的形式不一样，需要自己去灵活调整修改。

      • 小学做数学应用题时，会绞尽脑汗想怎么用算术来解

      • 初中做数学应用题时，学了多元方程式，发现之前的都是浮云

        此时方程式就是算法的概念：x，y就是变量，列出的等式就是算法的一种表达方式。

      • 到了高中，特别是省赛（我看了几套真题），发现通项公式是算法的一种表达方式。

        正如：1 — 100的求和的通项公式你一定熟悉

        • 实现方式一：定义首项、公差、项数变量，直接写一个表达式即可
        • 实现方式二：直接写1+2+3+…来实现
        • 实现方式三：定义起始位置、循环次数、步长，利用for循环实现

      孰好孰坏？显而易见。

具体安排建议

毕竟是参加省赛，不是玩编程。时间紧，任务重，必须要针对性的学习。 查阅了下资料，结合自己的看法，建议如下：

  1. 以老师的教程为主，这是前提。 老师是知道考试大纲的人，最专业。
  2. 完成老师布置的作业，再进行巩固。

    发现不懂的地方，即时咨询（谁都可以）

  3. 有条件的话，在家多上机实操。计算机类的东西，最忌讳只看书，最看重理论(30%)+实践(70%)相结合。

其他

  • 什么是省赛？ 答：高考作文，限时、有不小的难度。
  • 省赛和编程一样吗？ 答：不一样。省赛相当于应试教育，编程是兴趣技能。省赛会考一些稀奇古怪的知识点，但在实际项目中往往是用不到的。
  • 为什么要使用Dev C++ ？ 答：蓝桥杯等省赛强制使用dev，就是因为他没有代码提示。所以，要熟记语法。
